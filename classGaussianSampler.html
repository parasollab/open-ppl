Detailed Description

template<typename MPTraits>
class GaussianSampler< MPTraits >

Gaussian-distribution based sampling.

"The Gaussian sampling strategy for probabilistic roadmap planners" V.Boor, M.H. Overmars, A.H. van der Stappen.

The default behavior is the above standard gaussian sampling. However, this class provides internal (non-XML) functionality to perform gaussian sampling with other options.
