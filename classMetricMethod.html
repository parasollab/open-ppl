Detailed Description

Base algorithm abstraction for Metrics.

MetricMethod has one main function, operator() which returns a value associated with the roadmap for that metric. @usage

MetricPointer m = this->GetMetric(m_mLabel);
double v = (*m)(); //call as function object
double v2 = m->operator()(); //call with pointer notation
